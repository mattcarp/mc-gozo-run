# mc-gozo-run 🏃

Running app for the Gozo Half Marathon — live GPS tracking, spectator radar, voice alerts.

## Primary focus
Xagħra / Gozo Half Marathon 2026

## Features
- Live GPS route overlay (Xagħra route)
- Spectator radar — cheer team location markers  
- Voice alerts for pace, km splits, encouragement
- Race day dashboard
